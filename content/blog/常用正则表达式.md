## console filter

匹配一段使用字符串拼接 html, 输出拼接后的 html

```javascript
const r = /['"](.*)['"]\s?[\+;\b]?/g;
const template =
    "<div class='abcde'>" +
        "<div></div>" +
    "</div>";
cosnt output = template.replace(r, ($1) => $1)
//output = "<div class='abcde'><div></div></div>";
```

匹配一个匹配包括下划线, 连接线的任何单词字符并且不以 data 开头的, 以 true 结尾的字段
